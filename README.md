# JSTetris
Javascript implementation of the popular game Tetris

[Working Demo](http://elie-harfouche.com/Projects/Tetris.JS/)

## Features
* Base functions, rotate CW/CCW using left or right
* Next piece preview
* Score multipliers
* Space to drop piece 
* Shift to hold piece
* Leaderboard (WIP)
